# Welcome to Forsaken Casino

Forsaken Casino is an immersive VR memory card matching game built with A-Frame that challenges players to match pairs of cards within a 40-second survival timer. Set in a dark, atmospheric casino environment with flickering lights and eerie soundscapes, this horror-themed experience tests your memory skills while creating tension and urgency.

## What You'll Build

This project demonstrates a complete VR game implementation including:

- **3D Environment**: Casino table setup with atmospheric lighting and textures
- **Game Logic**: Card shuffling, flip detection, pair matching, and win/lose conditions
- **VR Interactions**: Gaze-based cursor controls for card selection
- **State Management**: Timer system, game state tracking, and board locking
- **Audio System**: Background music and interactive sound effects

## Technology Stack

- **A-Frame 1.4.0** - WebVR framework for building VR experiences
- **HTML5** - Game structure and asset management
- **JavaScript** - Game logic and interactivity
- **WebGL** - 3D rendering via A-Frame

## Project Structure

The game is contained in a single `index.html` file that includes:

- A-Frame scene setup with fog and lighting
- Asset preloading (images and audio)
- VR camera rig with gaze-based cursor
- Game board entity for card placement
- JavaScript game logic for mechanics and state management

<Note>
  Forsaken Casino runs directly in VR-capable browsers without requiring any build tools or dependencies beyond the A-Frame CDN.
</Note>
