## Overview

This page documents all configurable values, constants, and parameters used throughout Forsaken Casino VR. These values control gameplay mechanics, visual effects, timing, and environmental settings.

---

## Gameplay Configuration

### Timer Settings

<ParamField path="timeLeft" type="number" default="40">
  Initial time in seconds before the player loses their soul
  
  **Location:** index.html:60
  
  ```javascript
  let timeLeft = 40;
  ```
  
  **Behavior:**
  - Decrements every second during active gameplay
  - Displayed in the timer text as "VIDA: {timeLeft}s"
  - Game ends in defeat when reaches 0
  - Timer starts on first card interaction
</ParamField>

<ParamField path="timerInterval" type="interval" default="1000">
  Timer update frequency in milliseconds
  
  **Location:** index.html:71-75
  
  ```javascript
  timerInterval = setInterval(() => {
    timeLeft--;
    timerText.setAttribute('value', `VIDA: ${timeLeft}s`);
    if(timeLeft <= 0) die();
  }, 1000);
  ```
  
  **Configuration:** 1000ms = 1 second per tick
</ParamField>

  
  **Victory condition:** Game is won when `pairsFound === 4` (index.html:138)
</ParamField>

### Game State Variables

<ParamField path="gameActive" type="boolean" default="false">
  Tracks whether the game timer has started
  
  **Location:** index.html:62
  
  ```javascript
  let gameActive = false;
  ```
  
  **Transitions:**
  - `false` → `true`: On first card interaction (index.html:69)
  - Prevents timer from starting multiple times
</ParamField>

<ParamField path="lockBoard" type="boolean" default="false">
  Master lock preventing card interactions during processing
  
  **Location:** index.html:64
  
  ```javascript
  let lockBoard = false; // LLAVE MAESTRA PARA EVITAR TRABAS
  ```
  
  **Critical states:**
  - Locked when 2 cards are flipped (index.html:126)
  - Released after match/mismatch resolution (index.html:137, 151)
  - Prevents rapid clicking bugs
</ParamField>

<ParamField path="flippedCards" type="array" default="[]">
  Temporary storage for currently flipped cards
  
  **Location:** index.html:115
  
  ```javascript
  let flippedCards = [];
  ```
  
  **Behavior:**
  - Max length: 2 cards
  - Cleared after each pair evaluation (index.html:136, 150)
</ParamField>

## Visual Configuration

### Fog Settings

<ParamField path="fog.type" type="string" default="exponential">
  Fog calculation method
  
  **Location:** index.html:8
  
  ```html
  <a-scene fog="type: exponential; color: #0a0000; density: 0.15">
  ```
  
  **Options:**
  - `exponential`: Density increases exponentially with distance
  - `linear`: (not used) Density increases linearly
</ParamField>

<ParamField path="fog.color" type="color" default="#0a0000">
  Fog color in hexadecimal
  
  **Value:** `#0a0000` - Very dark red (almost black)
  
  **RGB:** R:10, G:0, B:0
  
  **Effect:** Creates ominous red-tinted darkness
</ParamField>

<ParamField path="fog.density" type="number" default="0.15">
  Fog thickness/opacity multiplier
  
  **Value:** `0.15` - Moderately dense fog
  
  **Range:** 0 (no fog) to 1 (very dense)
  
  **Visibility impact:** Objects become obscured at ~10-15 meters
</ParamField>

## Animation Timing

### Card Animations

<ParamField path="cardFlip.duration" type="number" default="250">
  Card rotation animation duration in milliseconds
  
  **Location:** index.html:122
  
  ```javascript
  card.setAttribute('animation', 'property: rotation; to: 0 180 0; dur: 250');
  ```
  
  **Rotation:** 0° to 180° over 250ms
</ParamField>

<ParamField path="cardFlip.textureDelay" type="number" default="125">
  Delay before changing card texture (mid-flip)
  
  **Location:** index.html:123
  
  ```javascript
  setTimeout(() => card.setAttribute('src', '#' + card.dataset.id), 125);
  ```
  
  **Timing:** Exactly halfway through flip animation (125ms of 250ms)
</ParamField>

<ParamField path="cardDisappear.duration" type="number" default="300">
  Matched card scale-to-zero animation
  
  **Location:** index.html:133-134
  
  ```javascript
  card1.setAttribute('animation', 'property: scale; to: 0 0 0; dur: 300');
  card2.setAttribute('animation', 'property: scale; to: 0 0 0; dur: 300');
  ```
  
  **Effect:** Cards shrink to nothing over 300ms
</ParamField>

<ParamField path="cardDisappear.delay" type="number" default="600">
  Delay before matched cards disappear
  
  **Location:** index.html:132
  
  ```javascript
  setTimeout(() => {
    card1.setAttribute('animation', 'property: scale; to: 0 0 0; dur: 300');
    // ...
  }, 600);
  ```
  
  **Purpose:** Allows players to briefly see the matched pair
</ParamField>

### Mismatch Animation Timing

<ParamField path="mismatchFlipBack.delay" type="number" default="800">
  Delay before mismatched cards flip back
  
  **Location:** index.html:142
  
  ```javascript
  setTimeout(() => {
    card1.setAttribute('animation', 'property: rotation; to: 0 0 0; dur: 250');
    card2.setAttribute('animation', 'property: rotation; to: 0 0 0; dur: 250');
    // ...
  }, 800);
  ```
  
  **Purpose:** Gives players time to memorize card positions
</ParamField>

<ParamField path="mismatchFlipBack.duration" type="number" default="250">
  Flip-back rotation duration
  
  **Location:** index.html:143-144
  
  **Rotation:** 180° back to 0° over 250ms
</ParamField>

<ParamField path="mismatchTexture.delay" type="number" default="125">
  Delay before changing texture back (mid-flip)
  
  **Location:** index.html:145
  
  ```javascript
  setTimeout(() => {
    card1.setAttribute('src', '#back');
    card2.setAttribute('src', '#back');
    // ...
  }, 125);
  ```
  
  **Nested timing:** 800ms + 125ms = 925ms total from initial flip
</ParamField>

## Customization Guide

<AccordionGroup>
  <Accordion title="Making the Game Easier">
    **Increase time limit:**
    ```javascript
    let timeLeft = 60; // Instead of 40
    ```

    ```javascript
    // Longer mismatch delay for easier memorization
    setTimeout(() => { /* flip back */ }, 1500); // Instead of 800
    ```
  </Accordion>

  <Accordion title="Making the Game Harder">
    **Decrease time limit:**
    ```javascript
    let timeLeft = 30; // Instead of 40
    ```

    ```javascript
    const cardData = ['img1', 'img1', 'img2', 'img2', 'img3', 'img3', 'img4', 'img4', 'img5', 'img5'];
    ```
    
    **Faster animations:**
    ```javascript
    setTimeout(() => { /* flip back */ }, 400); // Instead of 800
    ```
