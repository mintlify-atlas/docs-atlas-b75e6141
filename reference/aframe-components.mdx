## Overview

Forsaken Casino VR is built using [A-Frame 1.4.0](https://aframe.io/releases/1.4.0/aframe.min.js), a web framework for building virtual reality experiences. This page documents all A-Frame components and their configurations as implemented in the game.

---

## Scene Configuration

### a-scene

The root container for the VR experience.

<ParamField path="fog" type="object">
  Atmospheric fog configuration for the horror ambiance
  
  **Attributes:**
  - `type`: `exponential` - Creates dense, horror-themed fog
  - `color`: `#0a0000` - Deep red-black color
  - `density`: `0.15` - Controls fog thickness
</ParamField>

```html index.html:8
<a-scene fog="type: exponential; color: #0a0000; density: 0.15">
```

## Component Interactions

<Info>
  Components interact through JavaScript event listeners and attribute modifications. Cards listen for `mouseenter` events triggered when the cursor raycaster intersects them.
</Info>

### Event Flow

1. **Cursor raycaster** detects `.clickable` objects
2. **mouseenter** event fires on card hover
3. **JavaScript** modifies component attributes dynamically
4. **A-Frame** updates the 3D scene in real-time

```javascript index.html:103-110
card.addEventListener('mouseenter', function() {
  if (lockBoard || this.dataset.flipped === "true" || timeLeft <= 0) return;
  
  startTimer();
  new Audio(document.querySelector('#click-sound').src).play();
  flip(this);
});
```
