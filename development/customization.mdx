# Customization Guide

Forsaken Casino is highly customizable. This guide shows you how to modify every aspect of the game, from card images to atmospheric effects.

## Card Customization

### Adding More Card Pairs

The game currently uses 4 pairs (8 cards). To add more:

<Steps>
  <Step title="Add Images to Assets">
    Add new image elements in the `<a-assets>` block:
    
    ```html
    <a-assets>
      <img id="img1" src="img/img1.jpg">
      <img id="img2" src="img/img2.jpg">
      <img id="img3" src="img/img3.jpg">
      <img id="img4" src="img/img4.jpg">
      <!-- Add more pairs -->
      <img id="img5" src="img/img5.jpg">
      <img id="img6" src="img/img6.jpg">
      <img id="back" src="img/atras.jpeg">
      <!-- ... rest of assets -->
    </a-assets>
    ```
  </Step>
  
  <Step title="Update cardData Array">
    Modify the `cardData` array (source: index.html:65):
    
    ```javascript
    // Original (4 pairs)
    const cardData = ['img1', 'img1', 'img2', 'img2', 'img3', 'img3', 'img4', 'img4'];
    
    // Extended (6 pairs = 12 cards)
    const cardData = ['img1', 'img1', 'img2', 'img2', 'img3', 'img3', 
                      'img4', 'img4', 'img5', 'img5', 'img6', 'img6'];
    ```
  </Step>
  
  <Step title="Adjust Grid Layout">
    Modify the position calculation in `initGame()` (source: index.html:90-91):
    
    ```javascript
    // Original: 4x2 grid
    const x = (index % 4) * 0.4; 
    const y = Math.floor(index / 4) * -0.6;
    
    // For 6 pairs (3x4 grid)
    const x = (index % 3) * 0.4; 
    const y = Math.floor(index / 3) * -0.6;
    
    // For 6 pairs (4x3 grid - wider layout)
    const x = (index % 4) * 0.4; 
    const y = Math.floor(index / 4) * -0.6;
    ```
    
    <Info>
    Change `index % 4` to control columns, and the spacing `0.4` to control card separation.
    </Info>
  </Step>
  
  <Step title="Update Win Condition">
    Change the win threshold in `flip()` (source: index.html:138):
    
    ```javascript
    // Original
    if (pairsFound === 4) win();
    
    // For 6 pairs
    if (pairsFound === 6) win();
    ```
  </Step>
</Steps>

<Warning>
Increasing pairs makes the game harder! Consider increasing the timer duration accordingly.
</Warning>

## Timer Duration

### Adjusting Time Limit

Change the starting time (source: index.html:60):

```javascript
// Current: 40 seconds
let timeLeft = 40;

// Easy mode: 60 seconds
let timeLeft = 60;

// Hard mode: 25 seconds
let timeLeft = 25;

// Nightmare mode: 15 seconds
let timeLeft = 15;
```

<Warning>
The HUD initially displays "VIDA: 40s". Update the initial text element too:

```html
<!-- Update this to match your timeLeft value -->
<a-text id="timerText" value="VIDA: 60s" color="red"></a-text>
```
</Warning>

### Timer Speed Modification

Change countdown speed in `startTimer()` (source: index.html:71):

```javascript
// Current: 1 second per tick
timerInterval = setInterval(() => {
    timeLeft--;
    timerText.setAttribute('value', `VIDA: ${timeLeft}s`);
    if(timeLeft <= 0) die();
}, 1000);

// Half-speed (2 seconds per tick)
timerInterval = setInterval(() => {
    timeLeft--;
    timerText.setAttribute('value', `VIDA: ${timeLeft}s`);
    if(timeLeft <= 0) die();
}, 2000);

// Accelerating timer (gets faster over time)
let tickSpeed = 1000;
timerInterval = setInterval(() => {
    timeLeft--;
    timerText.setAttribute('value', `VIDA: ${timeLeft}s`);
    if(timeLeft <= 0) die();
    
    // Speed up every 10 seconds
    if (timeLeft % 10 === 0 && tickSpeed > 200) {
        clearInterval(timerInterval);
        tickSpeed -= 100;
        startTimer(); // Restart with faster speed
    }
}, tickSpeed);
```

### Changing Timer Display Text

Customize the HUD text format (source: index.html:73):

```javascript
// Current: "VIDA: 40s"
timerText.setAttribute('value', `VIDA: ${timeLeft}s`);

// English version
timerText.setAttribute('value', `TIME: ${timeLeft}s`);

// Dramatic version
timerText.setAttribute('value', `SOUL FADING: ${timeLeft}s`);

// Minutes:seconds format
const minutes = Math.floor(timeLeft / 60);
const seconds = timeLeft % 60;
timerText.setAttribute('value', `${minutes}:${seconds.toString().padStart(2, '0')}`);
```

## Animation Customization

### Card Flip Speed

Adjust flip animation timing in `flip()` (source: index.html:122-123):

```javascript
// Current: 250ms flip, texture swap at 125ms
card.setAttribute('animation', 'property: rotation; to: 0 180 0; dur: 250');
setTimeout(() => card.setAttribute('src', '#' + card.dataset.id), 125);

// Slow motion: 500ms flip
card.setAttribute('animation', 'property: rotation; to: 0 180 0; dur: 500');
setTimeout(() => card.setAttribute('src', '#' + card.dataset.id), 250);

// Lightning fast: 100ms flip
card.setAttribute('animation', 'property: rotation; to: 0 180 0; dur: 100');
setTimeout(() => card.setAttribute('src', '#' + card.dataset.id), 50);
```

<Info>
The texture swap timeout should always be half the animation duration for smooth flipping.
</Info>

### Match Animation

Change how matched pairs disappear (source: index.html:133-134):

```javascript
// Current: Scale to zero
card1.setAttribute('animation', 'property: scale; to: 0 0 0; dur: 300');
card2.setAttribute('animation', 'property: scale; to: 0 0 0; dur: 300');

// Fade out
card1.setAttribute('animation', 'property: material.opacity; to: 0; dur: 500');
card2.setAttribute('animation', 'property: material.opacity; to: 0; dur: 500');

// Spin and shrink
card1.setAttribute('animation__rotate', 'property: rotation; to: 0 360 0; dur: 600');
card1.setAttribute('animation__scale', 'property: scale; to: 0 0 0; dur: 600');
card2.setAttribute('animation__rotate', 'property: rotation; to: 0 360 0; dur: 600');
card2.setAttribute('animation__scale', 'property: scale; to: 0 0 0; dur: 600');

// Fly away upward
card1.setAttribute('animation__up', 'property: position; to: 0 5 0; dur: 800; easing: easeInQuad');
card1.setAttribute('animation__fade', 'property: material.opacity; to: 0; dur: 800');
card2.setAttribute('animation__up', 'property: position; to: 0 5 0; dur: 800; easing: easeInQuad');
card2.setAttribute('animation__fade', 'property: material.opacity; to: 0; dur: 800');
```

### Replacing Sound Effects


<Tip>
**Audio Format Recommendations:**
- Format: MP3 or OGG for best browser compatibility
- Sample rate: 44.1kHz
- Bit rate: 128kbps for music, 64kbps for SFX
- Length: Keep SFX under 1 second for snappy feedback
</Tip>
