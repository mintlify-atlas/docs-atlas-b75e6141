<Note>
Forsaken Casino is a WebVR memory card game built with A-Frame 1.4.0. Match all pairs before your life runs out to escape with your soul.
</Note>

## Getting Started

Follow these steps to get Forsaken Casino running on your device.

<Steps>
  <Step title="Download the Game Files">
    Clone or download the Forsaken Casino repository to your local machine:

    ```bash
    git clone <repository-url>
    cd forsaken-casino
    ```

    Ensure you have the following directory structure:
    ```
    forsaken-casino/
    ├── index.html
    ├── img/
    │   ├── img1.jpg
    │   ├── img2.jpg
    │   ├── img3.jpg
    │   ├── img4.jpg
    │   └── atras.jpeg
    ├── piso.jpg
    ├── lados.jpg
    ├── cartas.mp3
    └── fondo.mp3
    ```
  </Step>

  <Step title="Serve the Game Locally">
    The game requires a web server to load assets properly. Use any of these methods:

    <CodeGroup>
    ```bash Python
    # Python 3
    python -m http.server 8000
    ```

    ```bash Node.js
    # Using npx (no installation needed)
    npx serve
    ```

    ```bash PHP
    # PHP built-in server
    php -S localhost:8000
    ```
    </CodeGroup>

    <Warning>
    Do not open `index.html` directly in your browser. Asset loading requires an HTTP server.
    </Warning>
  </Step>

  <Step title="Open in Your Browser">
    Navigate to the game in your browser:

    ```
    http://localhost:8000
    ```

    The game will load with the A-Frame VR scene initialized automatically.
  </Step>

  <Step title="Start Playing">
    **Desktop Mode:**
    - Use your mouse cursor to hover over cards
    - Cards flip automatically on hover (`mouseenter` event)
    - The timer starts automatically when you flip your first card

    **VR Mode:**
    - Click the VR goggles icon in the bottom-right corner
    - Use your VR controller's laser pointer to select cards
    - Point at a card to flip it

    <Info>
    The game gives you **40 seconds** to match all 4 pairs (8 cards total). Match all pairs before time runs out to win!
    </Info>
  </Step>
</Steps>

## Game Mechanics

### How to Play

The game board consists of 8 cards arranged in a 4x2 grid on a virtual casino table:

```javascript index.html
const cardData = ['img1', 'img1', 'img2', 'img2', 'img3', 'img3', 'img4', 'img4'];
```

- **Objective:** Match all 4 pairs before the timer reaches 0
- **Timer:** You have 40 seconds of "VIDA" (life)
- **Flipping Cards:** Hover over cards to flip them
- **Matching:** If two flipped cards match, they disappear
- **Mismatching:** If cards don't match, they flip back after a delay

### Win Condition

Match all 4 pairs to see the victory message:

```javascript index.html
function win() {
  clearInterval(timerInterval);
  statusText.setAttribute('value', 'SALISTE CON VIDA');
  setTimeout(() => location.reload(), 4000);
}
```

### Lose Condition

If the timer reaches 0 before matching all pairs:

```javascript index.html
function die() {
  clearInterval(timerInterval);
  statusText.setAttribute('value', 'PERDISTE TU ALMA');
  deathOverlay.setAttribute('animation', 'property: material.opacity; to: 1; dur: 1000');
  setTimeout(() => location.reload(), 3000);
}
```

## Troubleshooting

### Cards Not Loading

Ensure all image files are in the correct directory:
```html index.html
<img id="img1" src="img/img1.jpg">
<img id="img2" src="img/img2.jpg">
<img id="img3" src="img/img3.jpg">
<img id="img4" src="img/img4.jpg">
<img id="back" src="img/atras.jpeg">
```

### Audio Not Playing

Browsers require user interaction before playing audio. The game handles this by starting audio when you flip your first card.

### VR Mode Not Working

See the [Requirements](/requirements) page for VR headset compatibility and WebXR browser support.
